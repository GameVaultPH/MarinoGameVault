How to Play

Launch Games With The Launcher

01

Connect The Drive

Plug the external drive directly into a USB 3.0 port and wait for Windows to show it in File Explorer.

02

Open The Launcher

Open the Marino Game Vault folder on the drive and run the Launcher. Choose Desktop Mode for mouse and keyboard, or Fullscreen Mode when using a controller.

03

Browse Your Library

Use the cover library, search, or filters to find a game. The included titles are already organized inside the Launcher.

04

Launch A Game

Select a cover and press Play. Keep the drive connected while playing and safely close the game before unplugging it.

05

Fix A Missing Path

If the Launcher cannot find a game after the drive letter changes, reconnect the drive or update the game's Play Action path and Working Directory to its folder.

06

Add A Missing Game

In the Launcher, choose Add Game then Scan Automatically and select the game's folder. You can also drag its executable into the Launcher window.

07

Common Fixes

Restart the Launcher in Safe Mode if it will not open. For game errors, update GPU drivers and install the included DirectX or Visual C++ components. Do not move individual game files.
